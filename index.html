            Setup Google Sheets Sync (One-time)

            1. Open Google Sheets and create a new blank spreadsheet
            2. Click Extensions > Apps Script
            3. Delete any existing code in the editor
            4. Open the file google-apps-script.js from this folder (it's in your iCloud Drive alongside this app), copy ALL the code, and paste it into the Apps Script editor
            5. Click the Save button
            6. Click Deploy > Manage deployments > Edit (pencil icon)
            7. Set Version to New version, click Deploy
            8. Copy the Web App URL and paste it below

            Then create a "Users" sheet in your spreadsheet with columns:
            pin | name | role
            Add a row for each user, e.g.: 1234 | Fatah | admin
